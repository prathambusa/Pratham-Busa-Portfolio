# Pratham Busa - Personal Portfolio Website

Welcome to my personal portfolio! This website showcases my professional journey, projects, and skills as a product manager and AI/ML engineer.

## 🌐 Live Site

Visit my portfolio at: [https://prathambusa.netlify.app](https://prathambusa.netlify.app)

## About This Portfolio

This is my personal website featuring:
- Modern, clean design with smooth animations
- Fully responsive layout for all devices
- Interactive project showcase
- Functional contact form with file attachments
- Auto-scrolling image carousels
- Professional email notifications

## Tech Stack

- **React** - UI framework
- **Vite** - Build tool and development server
- **TailwindCSS** - Utility-first CSS framework
- **Lucide React** - Icon library
- **Netlify** - Hosting and serverless functions

## 📝 Note

This is my personal portfolio website. While you're welcome to explore the code and use it as inspiration for your own projects, this repository represents my professional work and personal brand.

## 🚀 For Developers

If you're interested in the technical implementation or want to use this as a reference for your own portfolio, feel free to explore the codebase. The project demonstrates:

- Serverless function integration
- Modern CSS techniques with Tailwind

## 🛠️ Project Structure

```
src/
├── components/          # React components

│   ├── Projects.jsx    # Projects showcase

│   ├── Contact.jsx     # Contact form
│   └── Navigation.jsx  # Navigation component
├── App.jsx             # Main app with routing
├── main.jsx            # App entry point
└── index.css           # Global styles
```
